# csv-yt
Automated downloading of YouTube videos from CSV

## How To Install

Follow instructions at https://rustup.rs/

```shell
git clone git@github.com:z-fest/csv-yt.git

cd csv-yt

rustup update

cargo install --path .
```

## How To Use

```shell
csv-yt nominations.csv DV 'award' FL 'Team ' FV 'Team #' FL ' - ' FV 'Film Title'
```
